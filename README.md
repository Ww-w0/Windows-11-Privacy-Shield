# Windows 11 Privacy Shield 🛡️

A powerful batch script designed to enhance privacy and performance on Windows 11 by disabling intrusive features.

## ✨ Features
- **Remove Recall:** Completely disables the Windows Recall AI feature.
- **Stop Telemetry:** Disables data collection and background tracking services.
- **Privacy Tweaks:** Turns off Advertising ID and personalized suggestions.
- **Search Optimization:** Disables Bing web results in the Start menu.
- **Clean UI:** Removes "Recent Files" and "Frequent Folders" from File Explorer.
- **Performance Boost:** Cleans temporary files and stops unnecessary startup apps.

## 🚀 How to Use
1. Download `Ultimate_Privacy_Shield.bat`.
2. Right-click the file and select **Run as Administrator**.
3. Follow the on-screen prompts and **Restart** your PC.

## ⚠️ Disclaimer
This script modifies system settings and registry keys. Use it at your own risk. It is recommended to create a System Restore point before running.
